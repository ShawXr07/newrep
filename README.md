# Fastech - Sistema de Gestión de Aplicaciones Web
**Proyecto de Desarrollo de Software | Instituto Nacional de San Miguel Tepezontes**

Este proyecto consiste en una plataforma web integral diseñada para la gestión eficiente de datos. La aplicación utiliza un ecosistema moderno compuesto por **React.js** en el frontend, **Node.js** para la lógica del servidor y **MariaDB** como motor de base de datos relacional, ofreciendo una experiencia de usuario dinámica y adaptada a dispositivos móviles.

## Funcionalidades Implementadas

* **Arquitectura de Componentes:** Uso de componentes funcionales y Hooks para una gestión de estado eficiente.
* **Integración de API REST:** Desarrollo de endpoints en Node.js para la manipulación y consulta de datos en tiempo real.
* **Seguridad en Formularios:** Implementación de validaciones técnicas para asegurar la integridad de la entrada de datos.
* **Disponibilidad en Red Local:** Configuración del entorno para permitir el acceso y visualización desde diversos dispositivos en la misma red.

## Especificaciones Técnicas

### Configuración del Sistema
Para la correcta ejecución del software, se implementaron los siguientes requisitos:
* **Base de Datos:** MariaDB configurado en puerto local.
* **Entorno de Ejecución:** Node.js (v18 o superior).
* **Gestión de Dependencias:** NPM para la administración de librerías.
* **Configuración de Red:** Habilitación de tráfico en los puertos 3000 (Frontend) y 3001 (Backend).

### Arquitectura de la Aplicación
El sistema opera bajo un modelo **Cliente-Servidor**:
1. **Frontend:** Interfaz modular basada en React.js.
2. **Backend:** Servidor API en Node.js para procesamiento de peticiones HTTP.
3. **Persistencia:** MariaDB para el almacenamiento seguro de la información.

### Desafíos Técnicos Resueltos
* **Optimización de Consultas:** Configuración del controlador de MariaDB para reducir la latencia.
* **Protocolo CORS:** Configuración de políticas de origen cruzado para la comunicación entre el cliente y el servidor.
* **Adaptabilidad:** Diseño responsivo aplicado para asegurar la visualización en múltiples pantallas.
